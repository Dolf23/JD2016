1. Создайте папку для хранения Вашего кода
2. Имен папок соответсвуют заданиям на лекциях. Так, например задание по массивам из второй лекциии должно иметь название JD01_02.
3. Задания из рабочих тетрадей выполняются в папке JD01_work
4. ВАЖНО! Для корректного добавления (commit) и отправки на сервер (push) нужнен хотя бы один файл в каждой ВАШЕЙ папке.
5. СУПЕРВАЖНО! НИЧЕГО НЕ ПИШИТЕ В ЧУЖИЕ ПАПКИ!!! НИЧЕГО В НИХ НЕ МЕНЯЙТЕ. Все действия записываются!!!

6. После обновления не забудьте сделать pull request
